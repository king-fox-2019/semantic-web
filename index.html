Nav

  • It shows top level parts of your whole page map
  • As it implies, it should be used to help user navigate through your website.
  • And so in any page in your website, this part should look and stay the same.

Semantic HTML

Header

This part, in which the page title took place, is a semantic element called header. It serves as the title of the page.

Aside

This part usually placed on the left or right side (or both!) of the page flow. It serves almost like nav, but it's oftenly used to help user navigate in active page, and so could be different each page.

Main

This is the main part of your page. It holds almost every page-related contents of your web.

Article

This is oftenly used in an article page. As it's name implies, this element should hold main text part of your page.

Section

Usually, in the main element, you probably still need some separation for different context and parts. section serves this purpose of enabling you to separate your content to more cleaner and maintainable sections.

Footer

This is the last part of every page. It often holds general information about the website itself, contacts, copyrights, etc.
